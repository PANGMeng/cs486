1) In order to run the program for A5, extract the Learner.zip and ensure that the .txt and .csv files are in the same directory as the Learners.py file.

2) Open a command prompt and navigate to the location of the files, then run the following command:

	<testtype> = train or test, depending on if you want to test on the training data or the test data
